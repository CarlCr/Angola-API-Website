# Como Contribuir

Angola API é um projecto feito  por angolanos para desenvolvedores angolanos, por favor use de forma correta. Caso queira fazer parte deste grande projecto segue os passos abaixo.

1. Faça um Fork no projecto (https://github.com/Angola-Api/Angola-Api)	
2. Cria uma branch com a tua feature (`git checkout -b angolaapi/new-feature`)	
3. Faça um Commit com as tuas mudanças (`git commit -am 'Add new feature'`)	
4. Faça um Push para a Branch (`git push origin feature/new-feature`)	
5. Cria uma Pull Request
 
Caso deseja que uam funcionalidade seja adicionada no projecto, crie uma isses com o nome da feature que deseja e se possível detalhe sobre essa futura feature.
