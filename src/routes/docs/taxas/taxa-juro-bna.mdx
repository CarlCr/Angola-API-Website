# Taxa de Juros do Banco Nacional de Angola

Este endpoint retorna o valor da taxa de juros do BNA.

---

**GET** `https://angolaapi.herokuapp.com/api/v1/banks/rates/interest/bna`

 Exemplo da consulta
```json
// GET https://angolaapi.herokuapp.com/api/v1/banks/rates/interest/bna

{
  "rate": "15,5%"
}
```
