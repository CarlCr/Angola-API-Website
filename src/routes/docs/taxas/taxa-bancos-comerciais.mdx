# Taxa de variação das duas moedas estrangeiras mais populares de Angola

Este endpoint retorna o nome do banco, os dias e as taxas de juros de para depósitos a prazo efetuado pelos mesmos.

---

**GET** `https://angolaapi.herokuapp.com/api/v1/banks/rates/interest/term-deposit`

Exemplo da consulta

```json
// GET https://angolaapi.herokuapp.com/api/v1/banks/rates/interest/term-deposit

  "date": "25 de Fevereiro de 2021",
  "banks": [

          "rate": "3,50%"
        },

          "rate": "5,00%"

          "days": 360,
```
